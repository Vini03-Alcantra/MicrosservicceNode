# Microsservice NodeJS Example

## Projeto com 3 microsservices Client, Produto, ordered e o ordered consume dados dos serviços de Client e Produto e exibe informaçãoes

## Como rodar o projeto
### cd product 
### yarn dev

### cd client 
### yarn dev

### cd ordered
### yarn dev
